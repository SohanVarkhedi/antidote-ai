How It Works

Layered defense.
Real protection.

01
Data Poisoning Shield
Train-Time
Before a single sample reaches the model, ANTIDOTE's statistical outlier engine scores each data point. Anomalous data is quarantined — silent, automatic, tamper-evident.
02
Adversarial Hardening
Train-Time
Min-max robust optimization — the model learns to classify correctly even when an adversary actively tries to fool it. PGD-7 and FGSM augmentation cover every worst-case input.
03
Input Purification
Inference-Time
A denoiser strips adversarial perturbations before they reach the classifier. Feature squeezing reduces attack surface further. What arrives at your model is always clean.
04
Runtime Anomaly Detection
Inference-Time
A three-detector ensemble (IsolationForest + OneClassSVM + Z-score) scores every input. Suspicious inputs are flagged, rejected, or queued for review. Thresholds adapt dynamically.
