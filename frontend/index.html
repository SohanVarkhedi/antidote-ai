How It Works

Layered defense.
Real protection.

01
Data poisoning shield
Train-Time
Before a single sample reaches the model, ANTIDOTE's statistical outlier engine scores each data point. Cryptographic provenance hashes trace every sample's origin. Anomalous data gets quarantined — silent, automatic, tamper-evident.
02
Adversarial hardening
Train-Time
Min-max robust optimization during training — the model learns to classify correctly even when an adversary is actively trying to fool it. PGD-7 and FGSM augmentation create a model that has seen every worst-case input.
03
Input purification
Inference-Time
A diffusion-based denoiser strips adversarial perturbations before they reach the classifier. Feature squeezing reduces the attack surface further. What arrives at your model is always clean.
04
Runtime anomaly detection
Inference-Time
A parallel detector scores every input in feature space. Low-confidence, out-of-distribution, or perturbed inputs are flagged, rejected, or queued for review. Thresholds adapt dynamically to threat levels.
